INTRODUCTION AUX STATISTIQUES AVEC PYTHON
DESCRIPTION DU PROJET

Ce projet est un support pédagogique sous forme de notebook Jupyter destiné à
l’apprentissage des bases de la statistique à l’aide de Python.
Il combine des explications théoriques et des exemples pratiques afin de faciliter
la compréhension des concepts fondamentaux de la statistique descriptive.
Le projet s’adresse principalement aux étudiants débutants en statistique,
data science ou informatique.

CONTENU DU PROJET

genai.ipynb : notebook principal contenant le cours et les démonstrations
Modules abordés :

Module 1 – Introduction aux statistiques

- Définition de la statistique
- Notion de variable statistique
- Variables quantitatives et qualitatives
- Variables nominales et ordinales
- Population, échantillon et paramètres statistiques

Module 2 – Statistique univariée

- Description d’une variable qualitative
- Description d’une variable quantitative
- Premiers indicateurs statistiques
- Visualisation des données

TECHNOLOGIES UTILISÉES
- Python 3
- Jupyter Notebook
- Bibliothèques :
    - pandas
    - numpy
    - matplotlib
    - seaborn
  
INSTALLATION ET EXÉCUTION

  1- Télécharger ou cloner le projet

  2- Installer les bibliothèques nécessaires :
  pip install pandas numpy matplotlib seaborn

  3- Lancer Jupyter Notebook :
  jupyter notebook

  4- Ouvrir le fichier genai.ipynb

OBJECTIFS PÉDAGOGIQUES

  - Comprendre les bases de la statistique descriptive
  - Identifier et classifier les types de variables
  - Savoir décrire et analyser des données
  - Utiliser Python comme outil d’analyse statistique
  - Préparer l’apprentissage de la data science et du machine learning
